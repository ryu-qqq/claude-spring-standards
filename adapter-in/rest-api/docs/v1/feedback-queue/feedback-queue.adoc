=== FeedbackQueue 개요

피드백 큐를 관리하는 API입니다. LLM/Human 기반 2단계 승인 프로세스를 통해 코딩 규칙, 클래스 템플릿 등의 변경 요청을 관리합니다.

==== 상태 흐름

[source]
----
PENDING_LLM → LLM_APPROVED → (LOW: MERGED | MEDIUM: PENDING_HUMAN)
           ↘ LLM_REJECTED

PENDING_HUMAN → HUMAN_APPROVED → MERGED
             ↘ HUMAN_REJECTED
----

==== 리스크 레벨

|===
| 레벨 | 설명 | 승인 프로세스

| LOW
| 낮은 위험
| LLM 승인만으로 머지 가능

| MEDIUM
| 중간 위험
| LLM + Human 2단계 승인 필요

| HIGH
| 높은 위험
| LLM + Human 2단계 승인 필요 (추가 검토)
|===
