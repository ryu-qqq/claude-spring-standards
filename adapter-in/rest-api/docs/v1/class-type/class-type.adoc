=== ClassType 개요

클래스 타입 정보를 관리하는 API입니다. Aggregate, Value Object, Use Case 등의 클래스 타입을 등록/조회/수정할 수 있습니다.

ClassType은 ClassTypeCategory에 종속되며, 아키텍처별 클래스 타입 분류를 위해 사용됩니다.
