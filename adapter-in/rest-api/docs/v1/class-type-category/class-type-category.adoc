=== ClassTypeCategory 개요

클래스 타입 카테고리 정보를 관리하는 API입니다. 도메인 타입, 애플리케이션 타입, 어댑터 타입 등의 카테고리를 등록/조회/수정할 수 있습니다.

ClassTypeCategory는 Architecture에 종속되며, 아키텍처별 클래스 타입 분류 체계의 상위 계층입니다.
