=== Coding Rule 개요

코딩 규칙을 관리하는 API입니다. 레이어별 코딩 컨벤션, Zero-Tolerance 규칙 등을 등록/조회/수정/삭제할 수 있습니다.

==== 인덱스 조회 (경량)

규칙의 핵심 정보(code, name, severity, category)만 조회합니다.
캐싱 효율성을 위해 경량화된 응답을 제공합니다.
상세 정보가 필요하면 단건 조회 API를 사용하세요.
